The Project

Built for the love of CRPGs

Corundum Engine is a 2D CRPG engine being built from scratch in C++23, developed on Apple Silicon — in a genre that rarely is. It's early days, but the ambition is a complete forge: engine, tools, and game.

The dev log tracks progress as the engine, tools, and design take shape.

The Vision

What's Being Built

📦

Spritepacker

CLI tool that packs individual sprite images into texture atlas PNGs and companion JSON metadata — ready for the engine to consume.

Complete
🌎

Worldsmith

World map creation tool with native Azgaar's Fantasy Map Generator import. Paint terrain, place settlements, define states, and export tilemap chunks for the engine.

In Development
🗣️

Talesmith

Node-graph dialogue editor for branching conversations. Supports Talk, Choice, and Event nodes with conditions, variable mutations, sequencing, and event hooks. Quest editing planned.

In Development
🖼️

Spritesmith

ImGui-based editor for creating and managing sprite sheets, animation frames, and metadata directly within the engine's toolset.

In Development
🧱

Tilesmith

Visual tilemap editor for constructing 2D isometric worlds tile by tile — collision shapes, portals, layers, all integrated with the engine's renderer.

In Development
⚙️

2D Rendering

Hardware-accelerated 2D sprite and tilemap rendering via Sokol, with Metal, D3D11, and GLCore backends — targeting macOS, Windows, and Linux via GLFW.

In Development
⚔️

CRPG Systems

Rules systems, quest design, dialogue, combat, and character progression mechanics purpose-built for classic role-playing games.

In Development
📜

Data-Driven Design

Every aspect of the game — dialogue, tilemaps, sprite sheets, and configuration — is driven by JSON data. No hard-coded content, fully moddable.

In Development
🏰

Project Keystone

A 2D CRPG built entirely on Corundum Engine — the game the whole forge exists to ship. Story, world, and systems taking shape alongside the engine.
