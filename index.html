The Project

Built for the love of CRPGs

Corundum Engine is a custom 2D game engine written in C++23, designed specifically for building classic role-playing games. It uses Sokol for cross-platform hardware-accelerated rendering (Metal on macOS, D3D11 on Windows, GLCore on Linux) and GLFW for windowing. Every part of the engine — from the rendering pipeline to the RPG systems — is a work in progress.

The goal is a complete environment for developing story-driven 2D RPGs — engine, tools, and game, all built in-house.
