The Project

Built for the love of CRPGs

Corundum is a bespoke engine, built from scratch in C++23 to ship its first game: Project Keystone (codename), a 2D isometric fantasy CRPG. Engine, tools, and game grow together. One rule decides everything: C++ defines what can happen; JSON defines what does happen.

Isometric tilemaps, entity/component gameplay, branching dialogue, and staged quests are underway in the engine. Four in-house editors — Tilesmith, Spritesmith, Talesmith, and Spritepacker — author the JSON that drives it all.

Follow along as the engine, tools, and game take shape.
