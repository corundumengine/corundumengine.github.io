The Project

Built for the love of CRPGs

Corundum is a bespoke engine, built from scratch in C++23 to ship its first game: Project Keystone (codename), a 2D isometric fantasy CRPG. Engine, tools, and game grow together.

Isometric tilemaps, entity/component gameplay, branching dialogue, and staged quests are underway in the engine. Four in-house editors author the JSON that drives it all.

  • TilesmithTilemaps
  • SpritesmithSprite animation
  • TalesmithDialogue graphs
  • SpritepackerTexture atlases
